Capabilities

Everything you need, built in

Stitch comes packed with tools to handle your daily work, from managing your inbox to recording meetings and automating repetitive tasks.

Memory

Stitch remembers your preferences, workflows, and key facts across conversations. It learns what matters to you and applies that context automatically, no repetition needed.

Meeting recordings

Record meetings from Zoom, Google Meet, Teams, Slack, or Discord. Stitch captures audio locally, transcribes it, and generates summaries with action items, all on your machine.

Agenda & todos

A built-in task manager with priorities, due dates, and multiple lists. Stitch can create tasks from your conversations and meetings, so nothing slips through the cracks.

Daily automations

Turn any chat into a recurring workflow. "Summarize my emails every morning" or "clean up my downloads every Friday." Stitch runs it on schedule without being asked.

Email & calendar

Connect Gmail and Google Calendar. Stitch can read, draft, and send emails, manage your schedule, and handle routine communications.

Web browsing

Research topics, look up information, fill out forms, and pull data from websites. Stitch drives a real browser on your behalf.

MCP servers

Extend Stitch with any Model Context Protocol server. Add tools, data sources, and integrations from the growing MCP ecosystem — no plugin development needed.

Skills

Install community-built skills that teach Stitch new capabilities — from code review to data analysis. Plug in expertise on demand.

Installation

Getting started on your platform

Stitch is available for macOS and Windows. Your operating system may show a warning the first time you open it. Here's how to get past it.

macOS

  1. Download the .dmg file and open it
  2. Drag Stitch into your Applications folder
  3. Open Stitch from your Applications folder

Windows

  1. Download the .exe installer and run it
  2. Windows SmartScreen may show a "Windows protected your PC" warning
  3. Click "More info"
  4. Click "Run anyway"
  5. Follow the installer prompts to finish the setup

The warning appears because the Windows build isn't code-signed yet. We are working on it. Stitch is fully open source. You can inspect the code or build from source on GitHub.

Get started for free

Free API keys for LLM & Speech-to-Text

You don't need to spend a cent to use Stitch. These providers offer free tiers with generous limits — no credit card required.

Free LLM providers

Gemini AI Studio

  1. Go to aistudio.google.com/live and sign in with your Google account
  2. Click the API key button, which looks like a key
  3. Create a Google Cloud project, or import one if you already have it
  4. Click Create API key
  5. Copy the generated Gemini API key
  6. Paste the key into Stitch settings under the Google Gemini provider

Google AI Studio offers free Gemini API access with rate limits. No credit card is required to generate an API key. Usage is subject to Google's Gemini API terms and free tier limits.

OpenRouter

  1. Go to openrouter.ai and sign up with email, GitHub, or Google
  2. Open the top-right dropdown, then go to Preferences → API Keys in the sidebar
  3. Click New Key, give it a name, select No Expiration, and create it
  4. Copy the key — it's only shown once
  5. Paste the key into Stitch settings under the OpenRouter provider

Free tier includes 28+ models (DeepSeek R1, Gemma 3, Gemini Flash, and more) at 20 requests/min. No credit card needed. Use the :free model variant for zero-cost access.

NVIDIA NIM

  1. Go to build.nvidia.com and sign up for the free NVIDIA Developer Program
  2. Open your profile from the top-right menu and click API Keys
  3. Click Generate API Key, give it a name, and select Never Expire
  4. Copy your API key (starts with nvapi-) from the popup
  5. Paste the key into Stitch settings under the NVIDIA provider

Free tier gives you 1,000 inference credits on signup (up to 5,000 on request) with a 40 requests/min rate limit. Includes models like Nemotron, Llama, and DeepSeek. No credit card required. OpenAI-compatible endpoint.

Free Speech-to-Text provider

ElevenLabs

  1. Go to elevenlabs.io and create a free account
  2. Click Developers in the left sidebar
  3. Select the API Keys tab
  4. Click Create API Key, give it a name, and enable Speech-to-Text access
  5. Copy the key immediately — it's only shown in full once
  6. Paste the key into Stitch settings under the ElevenLabs STT provider

The free plan includes 10,000 credits/month which covers both Text-to-Speech and Speech-to-Text (Scribe). No credit card required. Used for transcribing meeting recordings in Stitch.

These providers are independent third-party services and are not affiliated with or endorsed by Stitch. Each provider has its own terms of service, privacy policy, and data handling practices. Please review the respective provider's website for details on how your data is used.

Privacy & trust

No middleman between you and your data

Stitch runs entirely on your computer. There are no Stitch servers, no accounts, and no tracking. When you use a cloud AI provider, your prompts are sent directly to that provider's API. For fully offline use, connect a local model instead.

Runs on your machine

Stitch itself is a desktop app with no backend. Your files, recordings, memories, and todos are stored locally. Nothing is ever uploaded to Stitch servers. There are none.

No accounts or tracking

No sign-ups, no telemetry, no analytics. Download the app and start using it immediately. Stitch doesn't know who you are and doesn't want to.

Direct-to-provider AI

When you use OpenAI, Anthropic, or Google, your prompts go straight from your machine to their API, not through us. Review each provider's data policies to understand how they handle your requests.

Fully offline option

Connect a local model and nothing leaves your computer at all. No internet connection needed, no data shared with anyone. Complete privacy for sensitive work.

Fully open source

Every line of code is public on GitHub. Audit what it does, build it from source, or contribute. No black boxes, no proprietary backends, no trust required.

You approve every action

Stitch asks for your permission before doing anything sensitive. You see exactly what it plans to do and can say no. You're always in the loop.
